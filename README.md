# Airport-ATS-System

Airport Air Traffic Control System implemented in C language.

## Project Description
This project simulates an airport air traffic control system (ATS) used to manage:
- Arrival flights
- Departure flights
- Emergency flights
- Passenger information

The system reads flight and passenger data from text files and allows the user to manage airport operations through a menu-driven interface.

## Files
- `P1_1231555_nadaTaleeb_section2.c` → Main source code
- `flights.txt` → Flights input file
- `passengers.txt` → Passengers input file
- `Project_Description.pdf` → Project requirements PDF

## Features
- Print flights
- Add new flights
- Land flights
- Depart flights
- Cancel flights
- Display flight status
- Add/remove passengers
- Emergency flight handling
